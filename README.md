# Fetch Contacts App

A simple app to fetch contacts 

Setup

## Steps include:

1. Clone the repo.

2. Install the dependencies :

    Install dependencies 
     ```bash
     npm install
     ```

3. Create a .env file in the root folder or rename .env_example to .env .

4. To run the app:

     
     ```bash
     npm start
     ```

    App runs on **http://localhost:30080** 



## Built With
ReactJs
